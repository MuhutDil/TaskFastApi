# TaskFastApi
A lightweight “to-do” application implemented on FastAPI using HTMX without using databases. You can add tasks in order and check the completed ones. You can delete unnecessary ones.

This is a training project based on an article [TestDriven.io](https://testdriven.io/blog/fastapi-htmx/).

#### Build the application and preparation for launch
    python -m pip install -r requirements.txt
#### Starting server
    uvicorn main:app
